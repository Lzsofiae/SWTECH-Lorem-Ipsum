    4. Analízis modell osztályai

      1. Main

        2. zsófika??? mire gondolt a költő?

      2. FileManagement

        3. Miután beérkezik a kép, továbbítja a Cloud-ba, vagy az Offline Storage-ba.

          Attribútum
        • pictureID: int - a feltöltésre kijelölt kép egyedi azonosítója.
          • Metódusok
        • getPicture() - Ezzel a metódussal olvassa ki, melyik képet kell feltölteni.
        • forwardPicture() - Amint megvan a kép, ezzel továbbítja.

      3. Offline Storage

        4. Ha nem lenne internethez kapcsolódva a telefon, akkor ide menti a feltölteni kívánt képet.

          Attribútumok
        • photo: file - Maga a kép ide kerül.
        • array: file[] - Ha esetleg több kép van, akkor létrejön egy tömb ezekhez.
          • Metódusok
        • getPicture() - Ezzel kapja magához a képet.
        • forwardPicture() - Amint online állapotba kerül a telefon, továbbítja a képet a Cloud felé.

      4. Cloud

        5. Ide kerülnek fel a képek a FileManagement-ből.

          Attribútumok
        • pictureID: int - A továbbított kép egyéni azonosítója.
        • dateAdded: date - Feltöltés dátuma.
        • sizeMB: double - Fájl mérete MegaByte-ban.
          • Metódusok
        • displayPicture() - Megjeleníti a képet a közösségi oldalon.
        • forwardResult() - Továbbítja mások értékelését a Results osztályhoz.

      5. PasswordManagement

        6. Új jelszó beállításához szükséges.

          Attribútumok
        • email: str - A felhasználóhoz tartozó email cím.
        • newPassword: str - Az új jelszó.
          • Metódusok
        • askNewPassword() - Ez egy dialógus, amiben a felhasználó beállíthatja a newPassword értékét.
        • setPassword() - Ezzel fogja rögzíteni a jelszavát.
        • encryptPassword() - Titkosítja a rögzített jelszót.

      7. WelcomePage

        8. Az alkalmazás üdvözlőképernyője. Innen tud majd az alany bejelentkezni.

          Attribútumok
        • userName: str - Felhasználói név.
        • next: button - Bejelentkezéshez használt gomb.
          • Metódusok
        • openLogin() - A bejelentkeztető folyamatot indítja el.
        • openHelp() - Segítség kérő ablakhoz irányít át.
        • getActualDate() - Bejelentkezés idejének rögzítéséhez szükséges.

      8. Help

        9. Alapvetően egy segítségkérő ablak. Az alanynak lehetősége van emberi segítséget kérnie.

          Attribútumok
        • helpEmail: str - A feladó email címe.
        • message: str - Maga az üzenet amivel az Adminhoz fordul.
          • Metódusok
        • setMessage() - Rögzíti az üzenetet.
        • sendMessage() - Elküldi az üzenetet.

      9. Login

        10. Bejelentkezéshez szükséges osztály.

          Attribútumok
        • userName: str - Felhasználói név.
        • password: str - Felhasználói névhez tartozó jelszó.
          • Metódusok
        • getUser() - Átadja a programnak a felh nevet.
        • getPassword() - Átadja a programnak a jelszót.
        • openRegistration() - Regisztrációs felületre irányítja át a felhasználót.

      10. Registration

        11. Regisztrálásra használt felület.

          Attribútumok
        • email: str - Felhasználóhoz rendelni kívánt email cím.
        • user: str - Felhasználó neve.
        • password: str Felhasználóhoz rendelni kívánt jelszó.
          • Metódusok
        • setEmail() - Rögzíti az email címet.
        • setUser() - Rögzíti a felhasználót.
        • setPassword() - Rögzíti a jelszót.

      11. UserMenu

        12. Belépés után erre az oldalra fog kerülni a felhasználó. Itt található majdnem minden felhasználói funkció. A jelszóváltást kivéve.

          Attribútumok
        • credit: int - Kreditjeinek száma.
        • rank: str - Kreditek alapján kiadott rang.
          • Metódusok
        • displayRank() - Megmutatja a rangot.
        • displayRating() - Megmutatja a krediteket.
        • openPhotos() - Megmutatja a feltöltött képeket.
        • openRating() - Megmutatja a besorolást.
        • openNews() - Generált híreket nyitja meg.
        • openSettings() - Beállítások megnyitásához szükséges.

      12. UserSettings

        13. Felhasználói beállításokat foglalja magába.

          Attribútumok
        • fontSize: int - Betűméret.
        • fontColor: int - Betűszín.
          • Metódusok
        • openPasswordManagement() - Jelszóváltó képernyőhöz visz át.
        • setFontColor() - Betűszínt állít be.
        • setFontSize() - Betűméretet állít.
        • logout() - Kijelentkezés

      13. Photos

        14. Képek feldolgozásával foglalkozik az osztály.

          Attribútumok
        • pictureID: int - Fénykép egyéni azonosítója.
        • foto: jpg - Fénykép fájl.
        • autoFocusOrNot: bool - Autofókusz használatával készült-e a kép vagy sem.
        • flash: bool - Használt-e vakut.
        • result: int - Képre visszaadott értékelések.
          • Metódusok
        • getPicture() - Kiválasztja a feltölteni kívánt fényképet.
        • displayResult() - Értékelést írja ki.
        • getFocusMethod() - Megállapítja a képről, hogy automata vagy kézi fókusszal készült.
        • getFlashUsed() - Megállapítja, hogy vakuval vagy anélkül készült-e.
        • forwardPicture() - Továbbítja a képet Offline Storage-ba vagy pedig Cloud-ba.

      14. Results

        15. Kiértékelés eredményét tárolja, továbbítja.

          Attribútumok
        • result: int - Összesített értékelés.
          • Metódusok
        • getResult() - Lekérdezi az összesített értékelést.
        • sendResult() - Elküldi a "result" változót.
